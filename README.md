# LeafScan Pi - Monitoreo de Salud de Cultivos de Tomate con Visión Artificial en Raspberry Pi

LeafScan Pi es un sistema de visión artificial diseñado para monitorear la salud de cultivos de tomate. 

Utiliza un modelo basado en MobileNetV2, entrenado con un subconjunto del dataset PlantVillage, y está optimizado para ejecutarse en tiempo real sobre una Raspberry Pi 3B+.
El sistema incluye una interfaz desarrollada en Streamlit que permite cargar imágenes, analizarlas y obtener recomendaciones automáticas para el manejo del cultivo.

Los modelos alcanzan una precisión cercana al 99%, demostrando un rendimiento adecuado para aplicaciones en campo.
